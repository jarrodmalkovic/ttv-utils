## Run Locally

### Clone

- Clone this repo to your local machine using `https://github.com/jarrodmalkovic/ttv-utils.git`

### Install server dependencies

```bash
npm install
```

### Install client dependencies

```bash
npm install
```

### Start server

```bash
npm start
```
